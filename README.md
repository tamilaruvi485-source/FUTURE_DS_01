# 📊 Future Interns – Power BI Task 1

## 👩‍💻 Project Overview
This project was completed as part of the **Future Interns Power BI Internship**. The dashboard provides a simple and interactive analysis of sales performance using the Sample Superstore dataset.

## 🎯 Objective
To create an interactive Sales Performance Dashboard using Microsoft Power BI and understand the basics of data visualization.

## 🛠️ Tools Used
- 📌 Microsoft Power BI
- 📊 Sample Superstore Dataset

## 📈 Dashboard Features
- 📊 Bar Chart – Sales by Category
- 🥧 Pie Chart – Sales by Region
- 📉 Line Chart – Sales by Order Date
- 💰 Total Sales Card
- 💵 Total Profit Card
- 🎛️ Region Slicer for Interactive Filtering
